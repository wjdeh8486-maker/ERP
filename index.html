📤 데이터 업로드

NAS의 엑셀 파일을 아래 해당 카드에 드래그하면 구글시트 파일 ID를 유지한 채 내용만 덮어쓰기합니다. API 주소 변경이 필요 없습니다.
사용 순서
1. NAS(\\Ntl\5.품질팀\...)의 엑셀 파일을 해당 카드에 드래그 (또는 클릭해서 선택)
2. 카드에 표시된 탭이 업로드한 엑셀의 같은 이름 탭으로 자동 교체됩니다 (엑셀에 없는 탭은 건너뜀)
3. 전부 완료되면 각 화면의 동기화(불러오기) 버튼을 눌러 최신 데이터를 반영하세요
※ 덮어쓰기는 탭 단위로만 이뤄지며, 같은 파일의 다른 탭은 건드리지 않습니다.
❓ 사용법 안내
🔍 검색 문법 (모든 검색창 공통)
쉼표(,)는 OR 검색입니다. 예: M200-005, M300-011 → 두 도번 모두 검색.
슬래시(/)는 AND(좁혀서) 검색입니다. 예: 유진/SHOWER → 유진 결과 중 SHOWER 포함만.
📋 관리항목 현황
검색해야 내용이 표시됩니다. 도번 카드의 [✕ 도번 제거]로 확인 끝난 도번을 전 구역에서 제외할 수 있고, [🗂 제거한 도번]에서 개별 복원, [↩ 전체 복원]으로 한 번에 되돌립니다. 제거 목록은 내 브라우저에만 저장됩니다 (다른 사람 화면에는 영향 없음).
월/년 생산 계획에서 월 헤더를 클릭하면 그 달 납기 항목만 상세에 표시됩니다. 납기 연장 권고의 간격은 도번별 과거 납품 이력에서 자동 학습됩니다.
⏰ 지연 LIST
지연일수 많은 순으로 상위 100건씩 표시되고 [▼ 더보기]로 추가 표시됩니다. 인쇄·엑셀은 항상 전체 기준입니다.
🔎 도번 이력 · 🚚 주간 출하
도번 이력: 도번/품명 검색 → 납품 이력, 지연 횟수, 학습 간격, 검사 이력 요약.
주간 출하: 이번 주·다음 주 최종납기 건을 요일별 보드로 표시 (아침 회의용).
🔄 데이터
상단 배너에 마지막 동기화 시각이 색으로 표시됩니다 (초록=10분 이내). 숫자가 다르면 새로고침으로 최신 데이터를 받아주세요. 인쇄 버튼은 남색, 검색·조회 버튼은 연노랑입니다.
